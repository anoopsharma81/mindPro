# Metanoia - Reflective Intelligence

## Brief Description

**Metanoia** is an AI-powered NHS appraisal assistant designed to help doctors streamline their annual appraisal process through intelligent reflective practice and CPD tracking.

### What It Does

Metanoia enables NHS doctors to:

1. **Create Structured Reflections**
   - Write reflections using the "What? So What? Now What?" framework
   - Automatically detect and analyze clinical cases using a specialized clinical reasoning framework
   - Map reflections to GMC Good Medical Practice domains (1-4)
   - Get AI-powered improvements and suggestions

2. **Track Continuing Professional Development (CPD)**
   - Record CPD activities (courses, reading, teaching, audits)
   - Auto-categorize activities by type and GMC domains
   - Track hours and generate impact statements
   - Link CPD activities to reflections

3. **Protect Patient Privacy**
   - Built-in PHI (Protected Health Information) detection
   - Warns users about potential patient identifiers
   - Ensures NHS Information Governance compliance

4. **Multiple Input Methods**
   - **Voice Notes**: Record and transcribe reflections on-the-go
   - **Document/Photo Extraction**: Snap photos of certificates, notes, or documents and extract structured reflections
   - **Manual Entry**: Type reflections directly

5. **AI-Powered Enhancements**
   - **Self-Play Improvement**: Iteratively improve reflection quality
   - **Clinical Reasoning Analysis**: Automatic analysis of complex medical cases with cognitive bias detection
   - **Auto-Tagging**: Intelligent CPD categorization and domain mapping

6. **Export & Appraisal Preparation**
   - Generate appraisal-ready documents
   - Organize by appraisal year
   - Export in multiple formats (DOCX, PDF, Markdown)

### Key Features

- ✅ **Year-Based Organization**: Organize reflections and CPD by appraisal year
- ✅ **GMC Domain Mapping**: Align all activities with Good Medical Practice domains
- ✅ **PHI Detection**: Automated patient information detection and warnings
- ✅ **Offline Support**: Works without internet connection
- ✅ **Secure Cloud Storage**: Firebase-backed with end-to-end encryption
- ✅ **Multi-Platform**: iOS, Android, and Web support

### Target Users

- NHS doctors preparing for annual appraisal
- Medical professionals tracking CPD activities
- Clinicians practicing reflective medicine
- Healthcare professionals seeking structured learning documentation

### Unique Value Proposition

**"Transform learning into understanding and reflection"**

Metanoia combines:
- **Structured frameworks** (GMC-aligned reflection templates)
- **AI intelligence** (clinical reasoning analysis, bias detection, quality improvement)
- **Privacy protection** (automated PHI detection)
- **Flexible input** (voice, photo, document, manual)
- **Seamless organization** (year-based, domain-mapped, searchable)

### Technology Stack

- **Frontend**: Flutter (iOS, Android, Web)
- **Backend**: Node.js with OpenAI integration
- **Database**: Firebase Firestore
- **Storage**: Firebase Storage
- **Authentication**: Firebase Auth (Email/Password, Google Sign-In)
